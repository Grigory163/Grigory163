### About Me

C++ developer and reverse engineer focused on Windows internals, kernel-mode programming, and binary analysis. Currently building kernel-level solutions for Windows software optimization — working with drivers, memory management, and system-level performance tuning.

I enjoy digging into how things work under the hood: from PE file structures and syscall mechanics to anti-debug techniques and code obfuscation patterns.

- Working on a **kernel-mode Windows optimization framework** — driver development, IRQL management, and system resource orchestration
- Exploring **binary analysis tooling**, custom disassembly engines, and static analysis pipelines
- Interested in **malware analysis**, **exploit development**, and **low-level security research**
